一、边框

1、半透明边框

这里可以看出背景会侵入到边框，平时看不出来，只有当边框为虚线时才看出

只有当设置background-color属性值为padding-box才不会侵入到边框

属性值有三个:

padding-box 背景被裁剪到内边距区域。背景将延伸到内边距的外边缘，并且不会被内边距覆盖。

content-box 背景被裁剪到内容框的边界。背景将仅绘制在内容框内，不会延伸到边框或内边距。

border-box 背景被裁剪到边框框线的内部。背景将延伸到边框的外边缘，并且不会被边框覆盖。

2、多层边框

1. box-shadow实现多层边框

box-shadow属性

box-shadow接受逗号分割的多个值

h-shadow：指定阴影水平偏移的长度。可以为负值表示向左偏移，正值表示向右偏移。

v-shadow：指定阴影垂直偏移的长度。可以为负值表示向上偏移，正值表示向下偏移。

blur：可选参数，指定阴影的模糊半径。值越大，阴影越模糊，值为 0 表示无模糊效果。

spread：可选参数，指定阴影的扩展大小。正值表示扩大阴影大小，负值表示缩小阴影大小。默认为 0，不扩展阴影。

color：可选参数，指定阴影的颜色。可以使用颜色值、十六进制码、RGB 值等。

inset：可选参数，用于指定阴影是否为内阴影。如果设置为 inset，阴影将出现在元素的内部。不设置或设置为其他值表示阴影在元素周围。

可以通过box-shadow属性设置多层外框，但是box-shadow实现的“边框”无法响应鼠标事件

但可以通过设置内嵌的box-shadow并设置相应宽的padding，来实现响应鼠标事件

而且阴影生成的边框不占据位置，不影响布局

2. outline实现多层边框

当谈到 CSS 的 outline 属性时，它用于在元素周围创建一个轮廓线，类似于边框但不占据空间。outline 属性可以用于为元素添加视觉效果，突出显示元素或为交互提供反馈。

outline-color：可选参数，用于指定轮廓线的颜色。可以使用颜色值、十六进制码、RGB 值等。默认值为当前文字颜色。

outline-style：可选参数，用于指定轮廓线的样式。可以取值为 dotted（点线）、dashed（虚线）、solid（实线）和其他样式。默认值为 none。

outline-width：可选参数，用于指定轮廓线的宽度。可以取值为像素（px）、百分比（%）、thin、medium、thick 等。默认值为 medium。

当谈到 CSS 的 outline-offset 属性时，它用于定义轮廓线与其相应元素的边缘之间的距离。
